# postie
A Reddit clone made for fun!

## Features
* User-created boards
* Posts
* Comment chains
* User profile pages for viewing recent posting history

## Technologies
### Postie.API
* C# 8
* ASP.NET Core
* EF Core 5
* Swagger
* SQLite as backing store

### Frontend
* Vue
* Axios

### etc.
* Auth0 for authentication
